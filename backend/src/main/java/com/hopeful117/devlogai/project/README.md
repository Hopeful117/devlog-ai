# Project Domain

## Purpose

The Project domain represents the software projects managed by DevLog AI.

A project is the root entity used to organize technical knowledge, development history, decisions, documentation and future analysis.

## Responsibilities

The Project domain is responsible for:

- Creating and managing projects
- Defining project identity
- Maintaining project lifecycle status
- Providing the foundation for future knowledge collection

## Current Features

Implemented:

- Project entity
- Project creation workflow
- Unique slug generation
- Project status management
- Persistence with PostgreSQL
- Database migration with Flyway
